# INF-001 — Network Outage

## Scenario
Regional or national network disruption affecting connectivity.

---

## Trigger Conditions
- ISP backbone failure
- Regional routing instability
- Large-scale packet loss detected

---

## Indicators
- High latency spikes
- Packet loss increase
- DNS resolution failures
- Multiple endpoint disconnects

---

## Threshold Logic

### GREEN
Stable connectivity

### YELLOW
Minor latency degradation

### ORANGE
Sustained connectivity degradation

### RED
Complete regional or systemic outage

---

## Expected Cascade

Network disruption  
→ Service unavailability  
→ API failure  
→ System downtime  
→ Business continuity impact

---

## Contingency Actions
- Switch to backup network routes
- Activate failover systems
- Notify users and stakeholders
- Prioritise critical services

---

## Recovery Actions
- Restore primary network routes
- Validate service integrity
- Gradual system reactivation
