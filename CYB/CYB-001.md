# CYB-001 — Ransomware Attack

## Scenario
Malicious encryption of systems causing operational disruption.

---

## Trigger Conditions
- Suspicious file encryption activity
- Endpoint detection alerts
- Network anomaly detection
- Unauthorized access patterns

---

## Indicators
- Spike in file modification events
- Security alerts from IDS/EDR systems
- Unusual outbound traffic
- Privilege escalation attempts

---

## Threshold Logic

### GREEN
No anomalies detected

### YELLOW
Suspicious activity under investigation

### ORANGE
Confirmed malware activity in isolated systems

### RED
Active ransomware spreading across systems

---

## Expected Cascade

Initial infection  
→ File encryption  
→ System downtime  
→ Service disruption  
→ Data access loss risk

---

## Contingency Actions
- Isolate affected systems
- Disable compromised accounts
- Activate incident response team
- Block network propagation
- Initiate forensic logging

---

## Recovery Actions
- Restore from clean backups
- Rebuild affected systems
- Rotate credentials
- Conduct post-incident review
