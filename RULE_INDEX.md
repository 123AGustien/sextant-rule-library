# Golden Rule Governance Layer

The Sextant Golden Rule applies across all domains:

## Principle

Protect system stability before optimizing performance.

## Governance Checks

- Prevent uncontrolled risk propagation
- Preserve critical system availability
- Maintain human oversight
- Prioritize containment before recovery
- Record every decision pathway

## Rule Application

Every simulation scenario must pass through the Golden Rule validation layer before:

- Risk escalation
- Cascade activation
- Solution recommendation
- Recovery execution
